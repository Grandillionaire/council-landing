# Council Landing

Minimal landing page for StartupCouncil AI advisory platform.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

Auto-deploys to Vercel on push to main branch.

## Design Principles

- Minimal, contemporary aesthetic
- Smooth animations with Framer Motion
- Glassmorphism and transparency effects
- Mobile-first responsive design
- Performance optimized
